설정이 필요해요

교사 구글 로그인과 교실 코드 기능을 쓰려면 무료 Firebase 설정값을 한 번 넣어야 합니다.

설정 방법 (5분)
  1. Firebase 콘솔에서 무료 프로젝트를 만듭니다.
  2. Authentication → Sign-in method에서 Google과 익명(Anonymous)을 사용 설정합니다.
  3. Firestore Database를 만들고, 아래 보안 규칙을 붙여 넣습니다.
  4. 프로젝트 설정 → 내 앱(웹)에서 firebaseConfig 값을 복사해, 이 파일의 FIREBASE_CONFIG에 붙여 넣습니다.
  5. 배포(Firebase Hosting 등) 후, Authentication → 설정 → 승인된 도메인에 주소를 추가하면 끝!
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /classrooms/{code} {
      allow read: if request.auth != null;
      allow create: if request.auth.token.firebase.sign_in_provider == 'google.com'
                    && request.resource.data.teacherUid == request.auth.uid;
      allow update, delete: if resource.data.teacherUid == request.auth.uid;
      match /scores/{uid} {
        allow read: if request.auth != null;
        allow write: if request.auth != null && request.auth.uid == uid;
      }
    }
  }
}
